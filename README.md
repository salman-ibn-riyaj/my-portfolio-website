## 📌 Project Overview

This is my personal portfolio website where I showcase my skills, projects, and experiences as a web developer. The site is built with **Next.js (App Router)**, styled with **Tailwind CSS**, and deployed on **Vercel** for fast, reliable performance worldwide.

Whether you're a recruiter, collaborator, or fellow developer — feel free to explore my work and get in touch!

---

## ✨ Features

- ⚡ **Blazing Fast Performance** — Powered by Next.js App Router with server-side rendering
- 📱 **Fully Responsive** — Optimized for all screen sizes (mobile, tablet, desktop)
- 🎨 **Clean Modern UI** — Minimal and professional design with smooth animations
- 🗂️ **Projects Showcase** — Highlights of my best work with live links and source code
- 🧑‍💻 **Skills Section** — Visual overview of technologies I work with
- 📬 **Contact Section** — Easy way to reach out directly
- 🌐 **SEO Optimized** — Meta tags and dynamic `generateMetadata` for better discoverability
- 🚀 **Deployed on Vercel** — Continuous deployment from GitHub
